## Project Overview

This repo is the focused Aquatic Motiv care-guides SEO/GEO app. It exists to publish and maintain a crawlable, mobile-friendly freshwater planted aquarium guide hub and supporting plant/invertebrate species pages.

The active public surface is the standalone care-guides origin:

`https://careguides.aquaticmotiv.com`

There is intentionally no Shopify-prefixed care-guide namespace and no default Shopify storefront routing integration.

Canonical plant care guides target:

- Title: `Aquarium Plants: Care Guides, Setup & Troubleshooting`
- URL: `https://careguides.aquaticmotiv.com`

Shopify integration rule:

- AquaticMotivApp is intentionally not plugged into Shopify storefront routing by default.
- Do not load Shopify MCP connectors, inspect Shopify routing settings, create Shopify pages/redirects, or attempt Shop integration unless the user explicitly asks for Shopify integration help in that turn.

## Current Scope

Keep the app simple:

- Care-guide hub linking to existing Shopify blogs/pages plus native species pages.
- Plant species pages, plant facet pages, and the Plant Finder quiz.
- Invertebrate species pages and invert facet pages.
- SEO/GEO metadata, JSON-LD, sitemap, and llms.txt routes.
- Shopify read-only price/stock enrichment when Storefront credentials are present.
- Validation and generation scripts for plant/invert data and hub cards.

Out of scope and intentionally removed: SaveAFish configurator, dashboard, auth, admin CMS, database/Prisma, generic fish/product/glossary/guide pages, API routes, notifications, maintenance tools, and mobile/PWA expansion.

## Data Flow

Content lives in JSON files under `src/data/`:

- `src/data/plants/species.json`
- `src/data/plants/article-links.json`
- `src/data/plants/care-page-links.json`
- `src/data/inverts/species.json`
- `src/data/inverts/article-links.json`
- `src/data/inverts/care-page-links.json`
- `src/data/guides/hub.json` generated by `scripts/gen_careguides_hub.py`
- `src/data/guides/page-card-images.json`

Data access functions live in `src/lib/data/`. The Plant Finder uses `src/lib/recommender/`. SEO schema helpers live in `src/lib/seo/schema.ts` and render through `src/components/seo/JsonLd.tsx`.

## Component Organization

- `src/app/` — all public standalone care-guide routes.
- `src/components/tools/` — hub, card, species, facet, and finder UI.
- `src/components/layout/CareHeader.tsx` and `CareFooter.tsx` — minimal standalone site chrome.
- `src/lib/shopify/` — Storefront read-only product fetch/normalization.
- `scripts/` — validation, hub generation, and Shopify metafield sync scripts.

## Validation

Before finishing code/data changes, run:

```bash
python3 scripts/validate-plants.py
python3 scripts/validate-inverts.py
npm run lint
npm run build
```
