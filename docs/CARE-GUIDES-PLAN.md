# Care-Guides Plan

Last updated: 2026-07-01.

## Current Decision

Use a native Shopify page for the care-guide hub. The old standalone careguides app is removed.

## Canonical URLs

- Public Shopify page: `https://aquaticmotiv.com/pages/freshwater-aquatic-planted-tank-guide`
- SEO-friendly redirect: `https://aquaticmotiv.com/freshwater-aquatic-planted-tank-guide`
- Old origin: `https://careguides.aquaticmotiv.com` is decommissioned as an application surface.

## In Scope

- Keep the Shopify page attractive, mobile-friendly, and SEO-focused.
- Keep the page source in `shopify/pages/freshwater-aquatic-planted-tank-guide.html`.
- Add internal links from ranking snail, shrimp, plant, and invertebrate articles back to the hub.
- Add consistent related-article blocks to existing Shopify blogs/pages after the link plan is reviewed.

## Out Of Scope

- Rebuilding the standalone Next.js app.
- Reintroducing Shopify app-proxy routing.
- Embedding the Plant Finder on the landing page.
- Adding new interactive tools without a separate scoped plan.

## Next Roadmap Step

Start with snail-based content because it already has ranking value. Add one or two body links back to the hub near the bottom of each relevant article, then add a consistent "If you like this article, try one of these" block with three or four related suggestions.
